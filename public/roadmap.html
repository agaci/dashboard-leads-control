Features planeadas

A plataforma YourBox Dashboard é uma solução sofisticada e madura para gestão de operações de entregas urgentes. Cobre o ciclo completo: captação de lead → cálculo automático → bot IA → pagamento → registo. A infraestrutura está preparada para crescimento, com pontos de integração já configurados para WhatsApp Business, pagamentos múltiplos e API pública.

Os itens de maior impacto imediato são: activação do webhook WhatsApp (70% da infra já existe), add-ons de serviço (aumento direto de margem) e testes automatizados (segurança no motor de preços).

Detalhe das Features Planeadas

1
WhatsApp Webhook — Completar Integração Evolution
A infra está 70% pronta: campos evolutionApiUrl, evolutionApiKey, evolutionInstance já configurados. Falta activar o webhook de entrada para processar mensagens recebidas no bot. Suportar: texto, imagem, localização, documento. Enviar templates HSM aprovados Meta.
Média Impacto Alto
2
Rastreamento em Tempo Real do Serviço
Página pública (sem login) onde o cliente acompanha a sua encomenda. Input: referência ou telefone. Mostra: localização GPS do driver em mapa, ETA, histórico de estados (recolhido → em transporte → entregue). Notificação WhatsApp/SMS quando driver se aproxima (10 min). Reduz contactos ao BO.
Média Impacto Alto
3
Add-ons de Serviço
Bot oferece serviços complementares após apresentar preço: embalagem (+3€), seguro para eletrónicos (+20€), SMS premium (+1€), agendamento de recolha (+2€). Colecção addOns editável no dashboard. Novo step OFFERING_ADDONS. Estimativa: +20% margem por lead.
Média Impacto Alto
4
Custom Pricing Rules
BO define regras de preço customizadas: "se cliente = ABC, +20% markup"; "se hora > 20h, +15% emergency surcharge"; "segunda-feira -5%". Colecção pricingRules com editor visual no dashboard. Aplicação automática em /api/price. Suporta contratos B2B e promoções sazonais.
Média Impacto Alto
5
Driver App Mobile
App iOS/Android para drivers: recebe serviços atribuídos em tempo real (push), navegação GPS integrada, foto de recolha/entrega com timestamp, assinatura do cliente, offline-first. Sem esta feature a operação não escala além de ~50 leads/dia. Requer mobile dev externo (React Native recomendado).
Alta Impacto Alto
6
IA Margens Dinâmicas
Algoritmo aprende padrões de leads convertidas vs perdidas. Input: rota, viatura, urgência, hora, reputação cliente. Output: desconto mínimo para fechar (não desconta 10% uniforme). Treino semanal offline. Endpoint de recomendação em tempo real (<200ms).
Alta Impacto Alto
7
Reputação de Cliente + Desconto Automático
Score 0-100 baseado em histórico: conversões, cumprimento de prazo, valor total, reclamações. Desconto automático 0-15% para clientes com score alto. Badge VIP visível no BO. Job diário recalcula scores. Ciclo virtuoso: cliente fidelizado → mais desconto → mais encomendas.
Baixa Impacto Médio
8
Bot com Visão IA (Análise de Imagem)
Cliente fotografa encomenda via WhatsApp. Claude Vision API detecta: dimensões estimadas, tipo de objecto, categoria de risco (frágil, perigoso). Sugere viatura automaticamente. Melhora acurácia do preço. Anthropic SDK já configurado no projecto.
Média Impacto Médio
9
Integração Contabilidade (Moloni)
Cada lead convertida gera fatura automática via Moloni API. Campos: cliente, NIF, origem, destino, preço, viatura, data. Envio PDF por email ao cliente. Exportação semanal de relatório de faturação. Conformidade fiscal PT obrigatória para crescimento.
Média-Alta Impacto Médio
10
Feedback Pós-Entrega
Email automático 7 dias após entrega: "Como correu?" com link para avaliação (1-5 estrelas + comentário). Rating <3 → escalamento automático para BO contactar. Dashboard com filtros por driver, cliente, rota. Alimenta sistema de reputação de cliente (feature #8).
Baixa Impacto Médio
11
Programa de Referência
Cliente gera código único (ex: JOAO2024AB). Novo cliente usa código → 5% desconto primeira encomenda. Referidor ganha crédito (+3€) por cada referência convertida. Dashboard: código, referidos, crédito acumulado. Aquisição viral com CAC baixo.
Baixa Impacto Médio
12
Testes Automatizados
Suite Jest + Playwright para motor de preços, fluxos bot e pagamentos. 20-30 casos críticos: 15 trajectórias Lisboa→Porto (comparação com Meteor), roteamento manual/auto por hora, fluxo INIT→LEAD_REGISTERED, mock Stripe webhook. CI/CD GitHub Actions antes de cada merge.
Média Impacto Alto
